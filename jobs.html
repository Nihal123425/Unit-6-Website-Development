Job Requirements - Hot Beans Web

Explore the skills, qualifications, and experience needed to join our team. We’re looking for passionate individuals ready to grow in web development.

Software Developer

Technical Skills

  • Java, Python, C#, JavaScript, PHP
  • React, Angular, Django, Spring
  • SQL and NoSQL databases
  • APIs and microservices
  • Git and cloud platforms

Soft Skills

  • Problem solving
  • Teamwork
  • Communication
  • Attention to detail

Education

  • Computer Science degree or related field
  • 1–5 years experience
  • Portfolio recommended

IT Support Specialist

Technical Skills

  • Hardware/software troubleshooting
  • Windows, macOS, Linux
  • Networking (DNS, VPN, DHCP)
  • Remote support tools
  • Basic cybersecurity knowledge

Soft Skills

  • Customer service
  • Communication
  • Patience and empathy
  • Time management

Education

  • IT diploma or equivalent
  • 1–2 years experience
  • Certifications (CompTIA A+ preferred)

Cybersecurity Analyst

Technical Skills

  • Firewalls, VPNs, IDS/IPS
  • Penetration testing
  • Security frameworks (NIST, ISO 27001)
  • Network security

Soft Skills

  • Analytical thinking
  • Attention to detail
  • Calm under pressure

Education

  • Cybersecurity or Computer Science degree
  • Certifications (Security+, CEH, CISSP)
