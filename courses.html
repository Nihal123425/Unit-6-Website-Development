Web Development Courses Available

Explore industry-leading courses designed to boost your skills and career in tech.

Udacity: Full Stack Web Developer

Overview: Project-based training for web developers. This course will teach you to build and deploy full-stack, database-backed APIs and web applications. Topics covered include identity access management, server deployment, containerization, data modeling, and API development.

This program uses 4 sub-courses, each with its own project (like a venue booking database or a trivia API). The capstone project involves building a full-stack web application that combines all of your new skills. Ideal for developers with background skills in Python, HTML, CSS, and Git.

Coursera: Introduction to Web Development

Overview: This course is great for beginners. It covers the basics of web development in 6 modules, including video lectures, quizzes, and hands-on practice.

You will learn how to structure and host a website, and gain foundational skills in HTML, CSS, and JavaScript. The course culminates in building a simple web application, with an introduction to form validation and JavaScript testing.

TripleTen: Software Engineering Bootcamp

Overview: A 10-month, part-time bootcamp that covers both front-end and back-end development, including JavaScript, React, Node.js, and more.

The course features project-driven learning, with real-world scenarios that culminate in a final project. There’s a robust career preparation module, including mentoring, externships, and post-graduation job support. With a money-back guarantee if you don’t land a job within six months, TripleTen offers an excellent pathway into tech.

Graduates have gone on to work at major tech companies like Tesla, Google, and Spotify.

Coursera: HTML, CSS, and JavaScript for Web Developers

Overview: This course covers the fundamentals of web development, including HTML, CSS, and JavaScript. It also teaches how to handle AJAX requests, process JSON data, and use the Bootstrap framework.

The course concludes with a capstone project where you’ll build a dynamic restaurant website. This is a great introductory course for those looking to gain foundational web development skills.

Scrimba: Frontend Developer Career Path

Overview: Focused on front-end development, this course covers HTML, CSS, JavaScript, React, async JavaScript, and more. It also includes UI design and working with APIs.

You will engage in code review sessions to improve your coding practices and receive feedback from industry professionals. Upon completion, you'll earn a Scrimba Frontend Developer Certificate. There is also a scholarship option available for those who need financial assistance.

Codecademy: Web Development Skills Path

Overview: In this course, you'll learn how to build websites using HTML, style them with CSS, and make them interactive with JavaScript. You’ll also dive into React for front-end development and Node.js with Express for back-end applications.

Additional topics include version control with Git, SQL basics, and test-driven development (TDD). By the end, you'll have built portfolio-worthy projects like a Yelp API search app and a website for a local coffee shop.
